# Dev environment for KC StartML Final Project

## Installation

- Install Docker Desktop:  
    https://www.docker.com/products/docker-desktop/  

## Run

1. Start project: 
    
    **Windows**
    ```
    run.cmd
    ```     
    **Linux/Mac**
    ```bash
    bash run.sh
    ```
2. Go to browser:  
    http://localhost:8080/
