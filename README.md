# Dev environment for KC StartML Final Project

## Installation

Install [Docker Desktop](https://www.docker.com/products/docker-desktop/)

## Run

0. Run Docker Desktop

1. Start project: 
    
    **Windows**
    ```
    ./run.cmd
    ```     
    **Linux/Mac**
    ```bash
    bash run.sh
    ```
2. Go to browser:  
    http://localhost:8000/
